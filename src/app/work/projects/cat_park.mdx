## Overview

Development of a flexible and highly customizable design system using React.js for front-end and Golang for back-end, and Postgres with Postgis for database

## Key Features

- **Component Library**: Built a set of modular, reusable UI components using React and styled-components, focusing on accessibility and responsiveness.
- **Theming and Customization**: Integrated a theming system that allows easy switching and customization of color palettes, typography, and layout styles using CSS variables.
- **Documentation and Usage Guidelines**: Developed comprehensive documentation in markdown to showcase components, usage patterns, and best practices, ensuring the design system is easy to adopt.

## Technologies Used

- **React.js**: For fast, server-rendered React applications.
- **Golang**: For creating api calls.
- **Bootstrap**: For styling React components with a modular, themable approach.
- **Postgres & Postgis**: For storing Vancouver Police Department's publicly accessible data including 534,506 records of crimes.
